### Commit style
Commit messages must conform to the to the [conventional commits format](https://www.conventionalcommits.org/en/v1.0.0/#summary).

```bash 
type(scope?): subject  # e.g app or comp
```

Message types:
- chore
- feat
- fix
- refactor
- revert
- style
- test
